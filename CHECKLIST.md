# Game Space "Clock" — Полный чек-лист продукта

---

## ✅ СДЕЛАНО

- Бронирование столика (FSM, уведомление менеджеру)
- День рождения (FSM, уведомление)
- Предложения пользователей
- Список игр PS5 / PS4
- Инструкции к настолкам с карточками-изображениями
- Фотогалерея (медиагруппы)
- Профиль + история броней + реферальный код (заготовка)
- Выбор языка UA / RU
- Постоянная кнопка "🏠 Меню" внизу
- Кнопка "◀️ Назад" во всех разделах
- Команды /start /help /myid в меню Telegram
- FSM guard — защита от нетекстовых сообщений
- Валидация даты брони
- Уведомления всем ADMIN_IDS
- Seed-скрипт для наполнения базы

## 🔧 НАВИГАЦИЯ И UX (приоритет)

- [ ] Анимированное приветствие при первом /start (фото заведения + текст атмосферы)
- [ ] Breadcrumb в заголовках: "🎮 Ігри → PS5"
- [ ] Кнопка "🔄 Рандомна гра" — случайная игра из каталога с описанием

- [ ] Пагинация списка игр (по 8 на страницу, кнопки «←» «→»)
- [ ] Поиск по названию игры (inline-mode или текстовый ввод)

## 🎮 ГЕЙМИФИКАЦИЯ (удерживает пользователей)

- [ ] **Бали (внутренняя валюта)**
  - Начисление: бронь +10, ДН +50, реферал +25, отзыв +5
  - Баланс в профиле с красивым прогресс-баром
  - История транзакций

- [ ] **Бонусний магазин**
  - Обмен баллов: бесплатный час, скидки, напитки, сувениры
  - Каталог с количеством баллов для каждого приза

- [ ] **Реферальна система**
  - Уникальная ссылка t.me/bot?start=REF_CODE
  - При регистрации по ссылке — оба получают баллы
  - Таблица лидеров рефералов (топ-10)

- [ ] **Досягнення / Бейджі**
  - "Перше бронювання", "5 відвідувань", "Король VR", "Майстер настолок"
  - Показываются в профиле
  - Push при получении нового бейджа

- [ ] **Рівні лояльності**
  - Bronze / Silver / Gold / Platinum
  - Повышают % скидки и приоритет броней
